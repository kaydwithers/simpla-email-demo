# Simpla CMS Demo

### To run locally
`http-server`

### Always pull to get any page edits
`git pull`

### To push changes
`git add.`
`git commit -m "Commit message"`
`git push -u origin master`

### Bower packages
`bower i simpla-text simpla-img simpla-admin simpla-article simpla-link simpla-collection --save`
